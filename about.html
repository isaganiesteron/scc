Speedcube Competitions (basic version)

What is this?

Speedcube competitions is a resource for organizers and competitors to know any information about a speedcube competition.

Why?

The primary purpose of this website is to help and give back to the speedcubing community. I have been part of the speedcubing community for many years and have organized a couple of competitions. While organizing competition I've found a lack of tools and resources that would greatly help with the process of organizing a speedcube competition. This website attempts to provide those tools and resources to help organize competitions which will help competitors as well.

My vision for speedcube competitions goes beyond just being a set of helpful tools. The main goal is to be able to sponsor and organize speedcube competitions to make competitions more frequent and available and also to organize and sponsor competitions with FREE registraion.

What does this do?

Currently the website is in a basic version.

Basic version:

  • Upload WCA competitor list file.
  • Automatically organize competitors into events and groups.
  • Change grouping of events
  • Assign roles, change groups of competitors
  • Add, Remove competitors
  • Save competition state to work on later
  • Save edited competition in CSV file

Complete version:

  • All features of the basic version
  • User Accounts
    • Allows organizers to create and save competitions.
    • Allows social media integration for a wider reach.
  • Competition pages for public viewing
    • Allows competitors to see their events, groups, roles, etc
    • Allow organizers to make announcements, update results live, make quick changes, etc

Who is this for?

This basic version is for organizers. The more complete version will cater to organizers and competitors of speedcube competions.

Monetization

This website will try to monetize by displaying ads and hopefully through sponsored deals.

How will the income of the website be used?

The main purpose of this website is to give back to the speedcubing community. There is an opportunity for monetization but it will be to give back as well.. Profits will be used in 2 ways. The 1 item is minimal and item 2 is the most important and exciting part.

  1. Website Overhead - Cover fees involved in running the website which include hosting fees and domain name fees.
  2. Speedcube Competitions - funding of, sponsorship, fund raising for equipment.
